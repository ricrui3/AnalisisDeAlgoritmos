* Para el archivo valores.txt el significado de cada valor es:
	n = cantidad de numeros a generar
	li = primer numero posible a generar
	ls = ultimo numero posible a generar

* El archivo numeros.txt es donde se escribiran los valores generados

* Para correr el programa de manera optima utilizar el siguiente comando una vez que se hayan elegido los valores esperados

	python generador.py < valores.txt > numeros.txt
