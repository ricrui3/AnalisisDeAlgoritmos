*Para ejecutar el algoritmo mergeSort es necesario ejecutar el sieguiente comando
	
	python mergeSort.py < numeros.txt > ordenados.txt

*Donde numeros.txt es el archivo con los números a ordenar creado con el generador
*En el cual la primer linea contiene el total de numeros a ordenar. ordenados.txt
*solo es el archivo donde se encotraran los numeros ya ordenados. En caso de querer
*generar un nuevo archivo numeros.txt correr el siguiente comando en la carpeta de 
*generador:

	python generador.py < valores.txt > ../MergeSort/numeros.txt
